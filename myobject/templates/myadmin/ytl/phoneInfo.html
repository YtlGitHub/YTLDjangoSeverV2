adb命令
获取序列号: adb devices
品牌: adb shell getprop ro.product.brand
设备名称: adb shell getprop ro.oppo.market.name
产品代号: adb shell getprop ro.separate.soft
产品型号: adb shell getprop ro.product.name
Android版本: adb shell getprop ro.build.version.release
OS: adb shell getprop ro.build.version.opporom
OTA版本: adb shell getprop ro.build.version.ota
IMEI_GUID_Carrier: adb shell dumpsys engineer --query_indicate_info
OTA开关(1为开，0或者空为关): adb shell getprop sys.ota.test
MTK/高通(打印MTK则是MTK，空为高通): adb shell getprop Build.BRAND
